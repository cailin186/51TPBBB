There are three kinds of feedback in this page: Good Evaluation, Bad Evaluation, Complaint.


Good/Bad Evaluation can be submitted only in students' APP and only when a student end the class; Complaint can be submitted both by students in APP and by CR/CC in CRM anytime.


Bad Evaluation and Complaints will be rated by four levels: L1, L2, L3, ZTP. The L3 and the ZTP will only show up in the Complaint, and will not show in teacher's page untill we judge the Validity to Valid.  The L1 and L2 both show up in Bad Evaluation and Complaints, and go directly to the teacher's page after 24 hours.


For the Good Evaluation, we need not to coach and validate them. The only operation we may need is to Remark if we think it's necessary.


For the Bad Evaluation, we need not to coach and validate them, either. So the default Coaching Status is Non Voice Coaching, and the default Validity is Valid. We can change the Validity to  Invalid if necessary, but we need not to change the Coaching Status here, it will be Non Voice Coaching all the time.


For the Complaint, if the level is L1 or L2, we need not to coach and validate them, and deal with them just like we deal with Bad Evaluation.


Only when the level of a Complaint is L3 or ZTP, we need to coach them. So when a L3/ZTP Complaint come into this table, the default original Coaching Status will be Un going, and the original default Validity is In Process which means we need to coach this complaint and to judge whether the complaint is Valid of Invalid. After this,we can change the Validity to Valid/Invalid, and the Coaching Status will change to Coached.


We can remark anything useful for a complaint by the function "Remark" all the time.


We can change the Validity and the type of  a Complaint/Bad Evaluation when we think it necessary, but only once: For frequent change may influence the statistical data.
