回填原SOURCE DETAIL的ID部分

• CHANNEL与SOURCE展示动态级联选项；CHANNEL更换后，SOURCE默认选中排序最前的选项

• 当且仅当CHANNEL为Reach时，SOURCE DETAIL可编辑，否则不可编辑且默认为0

• CHANNEL从别的选项切换为REACH时，SOURCE DETAIL默认回填本次编辑过程中的最新编辑内容

• 编辑SOURCE DETAIL时，仅支持输入数值（候选人/教师的系统ID）

• 保存时，须校验：

--CHANNEL为REACH时，SOURCE和SOURCE DETAIL不可为空，否则报错：source detail cannot be empty

--但SOURCE为REFFERR PROGRAMM和NON-EMPLOYEE时，SOURCE DETAIL可为空也可不为空，不做校验

--CHANNEL与SOURCE的对应关系正确匹配，否则报错：Channel does not match Source

--SOURCE DETAIL所填写的ID在系统中存在，否则报错：Nonexistent source detail ID

--SOURCE为Reach Upload时，SOURCE DETAIL ID必须也只能是教师的ID；（也只有教师的ID对应Reach Upload）否则报错：Source does not match Source Detail

--SOURCE为Applicant Referral时，SOURCE DETAIL ID必须也只能是候选人的ID；（也只有候选人的ID对应Applicant Referral）否则报错：Source does not match Source Detail
